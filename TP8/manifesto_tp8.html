RESUMO

O objetivo deste trabalho assenta na construção de um servidor, utilizando para o efeito o NodeJS, capaz de suportar um website de submissão e armazenamento de ficheiros.

Este website permite a submissão de um ficheiro e a visualização dos ficheiros já submetidos.

A aplicação construída é uma Single Page Application, onde todas as acções realizadas são tratados com recurso a AJAX e Jquery que, posteriormente, reencaminham a informação e os pedidos para o servidor (que escuta na porta 4008).

O projeto NodeJS foi gerido com o package manager do NodeJS permitindo guardar as dependências para os projetos permanentemente (pug,jsonfile,express,formidable e morgan). O servidor é corrido recorrendo ao comando npm test.

O rendering das páginas é feito com recurso a templates PUG e a uma base de dados dinâmica em JSON, que guarda todas as indicações dos ficheiros submetidos.

Na secção de resultados encontra-se um link para a pasta do TP8, onde se encontra a pasta fileStore, contendo a aplicação e os manifestos.
