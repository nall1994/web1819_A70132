RESUMO

O objetivo deste trabalho assenta na construção de um servidor, utilizando para o efeito o NodeJS, capaz de fornecer as páginas pedidas pelo cliente, sejam elas obras musicais, o indíce ou a própria folha de estilos.

O servidor deve receber pedidos para o servidor local (localhost) e devolver corretamente a página pedida.

O projeto NodeJS foi gerido com o package manager do NodeJS permitindo guardar as dependências para os projetos permanentemente.

O rendering das páginas é feito com recurso à linguagem PUG e a ficheiros de dados em JSON.

Tendo em conta que o dataset fornecido não continha um ficheiro de indíce, este foi gerado recorrendo a um pequeno programa em JAVA.
