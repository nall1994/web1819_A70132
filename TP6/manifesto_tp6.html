RESUMO

O objetivo deste trabalho assenta na construção de um servidor, utilizando para o efeito o NodeJS, capaz de suportar um website de uma lista de tarefas.

Este website permite o registo, listagem e remoção de tarefas.

O projeto NodeJS foi gerido com o package manager do NodeJS permitindo guardar as dependências para os projetos permanentemente (pug e jsonfile). O servidor é corrido recorrendo ao comando npm test.

O rendering das páginas é feito com recurso à linguagem PUG e a uma base de dados dinâmica em JSON.

Na secção de resultados aparecem apenas os três resultados principais do trabalho, sendo que os restantes ficheiros (pug e json) se encontram no repositório do Github.
